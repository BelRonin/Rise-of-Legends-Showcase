# Publication and Security Notes

## Public by Design

Этот репозиторий содержит только:

- описание продукта;
- подтверждённые возможности;
- высокоуровневую архитектуру;
- историю этапов разработки;
- план визуальных материалов;
- ссылку на публичный production-сайт.

## Intentionally Private

Здесь отсутствуют:

- исходный код production-приложения;
- SQL-миграции и полная модель данных;
- внутренние API-маршруты и правила доступа;
- environment variables;
- API keys, OAuth secrets и webhook secrets;
- данные участников и активные приглашения;
- IP-адреса и команды доступа к серверу;
- журналы административных и почтовых операций.

## Responsible Disclosure

Если вы обнаружили потенциальную проблему безопасности на production-сайте, не публикуйте технические подробности в публичном issue. Используйте официальный контакт, указанный на странице [контактов проекта](https://rise-of-legends.com/contacts).

## Source Availability

Production source code is private. Access may be provided selectively for a professional review under an appropriate confidentiality agreement.
