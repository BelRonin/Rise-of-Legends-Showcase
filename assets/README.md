# Visual Assets

В этом каталоге будут размещаться только проверенные презентационные материалы.

## Planned Screenshots

- главная страница: desktop и mobile;
- список команд и публичная карточка команды;
- расписание и турнирная сетка;
- кабинет капитана на тестовых данных;
- кабинет спикера на тестовых данных;
- административное управление без персональных данных;
- автоматическое продвижение победителя;
- официальный email-макет без активного приглашения;
- health-check и зелёный CI без внутренних журналов.

## Requirements

Перед добавлением каждого изображения необходимо:

- скрыть email, UUID, OAuth-параметры и токены;
- использовать тестовые данные в закрытых интерфейсах;
- закрыть лишние вкладки и уведомления браузера;
- не показывать настройки Supabase, Resend, VPS или GitHub secrets;
- подтвердить право использования имён и аватаров;
- проверить соответствие текущей production-версии.

Фиктивные скриншоты и изображения в showcase не используются.
